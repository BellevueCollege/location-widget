== Description ==
This plugin adds a widget under the Available Widgets list.
User can add this plugin to list their location information. Following are the fields that the widget offers:
Location Widget Title
Select - Builiding Pic
Office Location
Hours Title
Office Hours

== Installation ==
1. Download plugin files from https://github.com/BellevueCollege/location-widget.
2. Place the plugin folder in your `wp-content/plugins/` directory and activate it.
3. You will see Widget Name Office location under the widget list.
